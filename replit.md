# Portfolio Site — Cid Kageno

## Overview
A modern, animated personal developer portfolio for Cid Kageno, a Full Stack Developer and AI Systems Builder. Features a dark aesthetic with ambient animated backgrounds, scroll-triggered reveals, a typewriter hero, and an integrated "Ani Chat" AI assistant.

## Design System
- **Theme**: Dark-first with purple/cyan gradient accents (`#7c6af7`, `#06b6d4`)
- **Typography**: Inter (UI) + JetBrains Mono (code/labels)
- **Animations**: Floating orb background, spinning avatar ring, typewriter effect, scroll reveal, glow effects
- **Components**: Glassmorphism navbar, gradient cards, animated stats, pill-style tech stack badges

## Tech Stack
- **Frontend**: Pure HTML5, CSS3, Vanilla JavaScript (ES6+) — no build step
- **Backend**: Minimal Node.js (`server.js`) — static file server + GitHub repos proxy
- **Auth**: AWS Amplify Gen 2 (`aws-amplify/auth` via CDN) — Cognito User Pool
- **AI Chat**: Ani chat at `/Ani/` calls `https://ani-jms7.onrender.com/api/chat` directly
- **Data**: GitHub API proxied via `/api/github-repos` (frontend falls back to direct API)
- **Firebase**: `firebase-config.js` kept hardcoded for Firebase Realtime Database access

## Auth Architecture (Amplify Gen 2 Serverless)
- `amplify/backend.ts` — Amplify Gen 2 entry point using `defineBackend`
- `amplify/auth/resource.ts` — `defineAuth` with email + Google OAuth (Cognito)
- `auth.js` — Frontend auth helpers using `aws-amplify/auth` via esm.sh CDN
- `amplify_outputs.json` — **Placeholder** — auto-generated by `npx ampx sandbox` on deploy
- Both auth modals (main site + Ani) handle: Sign In, Sign Up, Email Verification code

## Running Locally on Replit
```
node server.js
```
Serves on port 5000. Auth UI shows but auth calls require a deployed Amplify backend.

## Deploying to AWS Amplify
1. Push repo to GitHub/GitLab
2. Connect to AWS Amplify console
3. Set Amplify secrets: `GOOGLE_CLIENT_ID`, `GOOGLE_CLIENT_SECRET`
4. Update callback URLs in `amplify/auth/resource.ts` to your Amplify domain
5. Amplify auto-runs `amplify.yml` which generates `amplify_outputs.json`

## Environment Variables
- `GITHUB_TOKEN` — (optional) GitHub API token for higher rate limits

## User Preferences
